# 高驰运动数据分析

用于分析 COROS 高驰手表用户训练数据的 AI Skill。

## 能力

- 🏃 **运动记录分析** - 各类跑步数据，支持分段拆解
- 📊 **训练负荷评估** - ShortTerm/LongTerm Load 分析
- 😴 **睡眠恢复追踪** - 睡眠质量、HRV、恢复状态
- 🎯 **高驰预估** - VO2max、阈值配速/心率、预估全马/半马
- 📈 **配速区间识别** - E/M/T/I/A 区配速与心率对应
- 📅 **自然周跑量统计** - 按自然周汇总，历史 8-12 周分析

## 数据处理规则

- 运动记录无时分秒，同日多次运动需用户确认时段
- 睡眠日期为醒来当天，非入睡当天
- 自然周跑量按 [100,101,102,103] 计入（户外跑/室内跑/越野跑/操场跑）
- 不完整周不下结论，等周末数据补齐后判断

## 适用场景

"分析最近一个月训练" / "我的马配和乳酸阈值是多少" / "预估一下全马成绩" / "近8周周跑量" / "分析这次间歇课分段数据"
