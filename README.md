# 高驰运动数据分析

用于分析 COROS 高驰手表用户训练数据的 AI Skill。

## 能力

- 🏃 **运动记录分析** - 运动汇总数据（均配速、最佳1km配速、心率、步频等）
- 🌤️ **天气数据关联** - 自动采集训练当天温湿度，结合不适指数（DI）量化分析
- 📊 **训练负荷评估** - ShortTerm/LongTerm Load 分析
- 😴 **睡眠恢复追踪** - 睡眠质量、HRV、恢复状态
- 🎯 **高驰预估** - VO2max、阈值配速/心率、预估全马/半马
- 📈 **配速区间识别** - E/M/T/I/A 区配速与心率对应
- 📅 **自然周跑量统计** - 按自然周汇总，历史 8-12 周分析
- 🔧 **自动修复** - 未配置 MCP 时自动尝试修复配置
- 🔐 **自动授权** - 无可用 token 时自动引导 COROS 账号授权

## 天气分析说明

- 通过 Open-Meteo 免费 API 获取训练当天天气数据
- 计算不适指数（DI）：`DI = T - 0.55 × (1 - 0.01 × H) × (T - 14.5)`
- DI 阈值：< 17 最佳 / 17-19 轻度影响 / > 19 明显影响
- 温湿度量化：每+1°C 约+5s/km，每+10%RH 约+3-5s/km
- **核心原则：疲劳 > 天气**（天气解释力 < 10%，疲劳累积解释力 > 80%）

## 数据处理规则

- 运动记录无时分秒，同日多次运动需用户确认时段
- 睡眠日期为醒来当天，非入睡当天
- 自然周跑量按 [100,101,102,103] 计入（户外跑/室内跑/越野跑/操场跑）
- 不完整周不下结论，等周末数据补齐后判断

## 适用场景
